# kaneo-cli

CLI for [Kaneo](https://kaneo.app/) project management.

## Install

```bash
npm install -g kaneo-cli
```

## Supported Platforms

| Platform | Architecture |
|----------|-------------|
| Linux | x86_64 |
| macOS | x86_64, Apple Silicon |
| Windows | x86_64 |
